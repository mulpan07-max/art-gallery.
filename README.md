The Art Gallery
A Digital Tribute to Renaissance Masterpieces

An elegant, responsive website showcasing Renaissance art exhibitions with a modern museum-inspired design. 

Description
The Art Gallery is a visually immersive web experience that brings the grandeur of Renaissance art into the digital space. Designed with attention to typography, color harmony, and spatial rhythm, the site invites visitors to explore timeless masterpieces as if walking through a curated museum hall.

Key Features:
Hero section with atmospheric background and poetic typography
Interactive gallery featuring layered image cards with hover effects
Ticket booking system with a refined, glass-morphism form
Contact & info section blending historical quotes with contemporary layout
Smooth animations and micro-interactions for enhanced user engagement
 Technologies Used
HTML5 — semantic structure
CSS3 —
Custom Properties (CSS Variables)
Flexbox & CSS Grid for layout
clamp(), min(), max() for fluid typography
Media Queries for full responsiveness
backdrop-filter for glass-morphism effects
Bootstrap Icons — for intuitive navigation
Google Fonts — Playfair Display (serif elegance) + Inter (clean readability)
  Fully Responsive
Mobile-first approach
Adapts seamlessly to phone, tablet, and desktop
Touch-friendly buttons and form fields
Safe-area support for iPhone notch devices
